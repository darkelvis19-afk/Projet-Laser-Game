LaserGame – Jeu Shoot’em Up 2D

LaserGame est un jeu 2D de type Shoot’em Up développé en Java avec LibGDX.
Le joueur contrôle un avion et doit survivre le plus longtemps possible face à des vagues d’ennemis.

Prérequis

Gradle (fourni avec le projet)

Compilation et Exécution

Windows
.\run.bat

Linux / Mac
chmod +x run.sh
./run.sh

Alternative (Gradle)
./gradlew :desktop:run

Contrôles du Jeu

Flèches directionnelles ou W A S D : déplacer l’avion

Espace ou Flèche Haut : tirer

P : mettre en pause

Souris : navigation dans les menus
